# algorithm-study

### 📒 스터디 규칙
- 1주일에 최소 Day 5개 이상 문제 풀이
- 문제 풀이는 개인 브랜치에서 진행하며, 완료되면 PR 올리기
- 시간 날 때마다 자유롭게 PR 리뷰 참여
- 일요일에는 밀린 PR 리뷰를 작성하며 본인을 제외한 나머지 두 명의 모든 문제에 대해 리뷰

---

### 🔒 리뷰 작성 가이드
아래 중 하나 이상을 포함하여 작성한다.

- 접근 방식에 대한 의견
- 시간 / 공간 복잡도 관련 피드백
- 더 단순하거나 효율적인 풀이 아이디어
- 코드 가독성 및 구조 개선 제안

특별히 지적할 부분이 없는 경우에도 다음과 같은 내용으로 리뷰를 진행한다.
- 잘한 점에 대한 칭찬
  - 문제 접근 방식
  - 코드 구조
  - 변수명, 예외 처리 등
- 인상 깊었던 포인트 언급
